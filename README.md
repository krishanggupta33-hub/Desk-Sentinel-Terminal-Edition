It is a typing game 

Codes fall from the top of the screen. You type the code before it hits the bottom or your server takes damage. Clear enough blocks and it speeds up / spawns faster. Health hits 0, game over. The game is based on python.

Install & run (Windows)

This is a desktop game

Download the release from Github 

Unzip the file and play
